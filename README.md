# TrueMark — Smart Attendance Management App

TrueMark is an Android app that digitizes classroom attendance with 
role-based access for Admins, Faculty, and Students.

## Features
- Role-based dashboards for Admin, Faculty, and Student
- QR-code based attendance sessions for quick, accurate check-ins
- Firebase Authentication, Firestore, and Realtime Database for live sync
- In-app chat between faculty and students
- Push notifications via Firebase Cloud Messaging
- Offline-friendly local caching using Room database
- Attendance history and reporting

## Tech Stack
- Kotlin
- Firebase (Auth, Firestore, Realtime Database, Cloud Messaging)
- Room Database
- Jetpack Compose / XML with View Binding
